My microbit project

I worked on coding microbit and its servo extesion. I used microbit to control the movements of the servo. I worked with Jason for some time to figure out how to connect servo to microbit. I then wrote a code that is for controlling servos as propellers to move an airship. It allows users to use microbit buttons to control the movement of 2 servos.

Some things I needed to do are:

  • test out how the codes will affect the servos
